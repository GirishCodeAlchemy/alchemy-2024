### **_variables_**

- `let` to make a constant (immutable)
- `var` to make a variable (Mutable)

### **_Datatypes_**

**_TypeInference_**

    - let myAge = 35 //Integter
    - let myName = "Girish " //Stign
    - let temp = 87.5 //floating point
    - let isValid = true // Boolean

**_type annotation_**

    - let friend: String = "Girish "
